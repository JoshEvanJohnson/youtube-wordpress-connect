== Installation ==

1. Download plugin and drop it into your plugins directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place the shortcode in the desired page or post.

[ytwpconnect channel="soccer" showcase="gIHCVA54XnE" limit="6" showcase_width="630" thumbnail_width="200"]
